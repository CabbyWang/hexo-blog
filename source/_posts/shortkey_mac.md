# mac常用快捷键(持续更新)

- 截屏
   - 全屏幕截屏 `command + shift + 3`
   - 手动截屏  `command + Shift + 4`

- 最小化
`command + m`

- 最大化
`command + ctrl + f`

- 关闭当前窗口
`command + w`

- 关闭整个应用
`command + q`

<!--more-->

- 光标移动
   - 一行最前（home） `fn + left` or `control + a`
   - 一行最后（end） `fn + right` or `control + e`
   - 最前 `fn + top`
   - 最后 `fn + bottom`
 
- 打开用户目录
`shift + command + h`

- 显示隐藏文件/文件夹
`command + shift + .`

- 前往任何文件夹
`command + shift + g`
