# React 学习之路
npm安装慢， 可以切换淘宝源
```bash
npm config set registry https://registry.npm.taobao.org
```
构建react工程
```bash
npm install create-react-app -g    # 全局安装react
create-react-app hello-react       # 创建react工程
cd hello-react                     # 切换到工作目录
npm start                          # 启动工程
```
